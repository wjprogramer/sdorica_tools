# Sdorica

## Usage

1. 裝 [Node JS](https://nodejs.org/en/)
2. 在 `sdorica_tools` 資料夾底下打開終端機
3. 輸入指令 `npm install`
4. 輸入指令 `npm run start`
5. 打開瀏覽器，進入網址 `localhost:3000`
